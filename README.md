# PRODIGY_DS_02
Task 02-  Relationship between variables identify patterns and trends in the data

🚀 Completed Task 02 of my Data Science Internship at Prodigy InfoTech !

📊 Task: Exploratory Data Analysis (EDA).

🛠️ Tools Used:

Pandas, Python, Matplotlib, Seaborn, Google Colab.

📌 What I did:
Loaded and importing the dataset.
Dropped columns (cabin)
Analyze relationship & variables
Created multiple visualizations(Fare,Age,Gender,Class.

💡 Key Insights:
Gender and class were the strongest predictors of survival
Females and 1st class passengers had higher survival rates
Higher fares increased survival probability
Children were more likely to survive
No strong relationships between age & fare
